. Obsfurcação de código

Técnicas usuais para esconder códigos de linguagens interpretadas.

Escrito em: 24/01/2022

. O que é

A obsfurcação de código visa, a partir de um código escrito em uma linguagem interpretada, gerar um outro código equivalente, mas de leitura muito mais difícil.

Essa técnica é muito comum para camuflar códigos escritos em JavaScript usados em aplicações web. Com o código obsfurcado, é consideravelmente mais difícil realizar engenharia reversa e procurar por falhas de segurança.

. Ferramentas

Já existem diversas ferramentas prontas para obsfurcar código. Você pode encontrar exemplos e a documentação de como usar nos sites de cada uma dela. Algumas delas são:

  • obfuscator.io
  • jsfuck
  • jjen code
  • aaencode

. Desobsfurcação

Também existem diversas ferramentas para desobsfurcar código, aqui citamos duas:

  • O jsconsole é um emulador de console de JavaScript e permite ver o comportamento do código obsfurcado.
  • O jsnice tenta desobsfurcar o código automaticamente.
